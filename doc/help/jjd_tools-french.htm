Module JJD pour XOOPS

Ce module n'est pas vraiment un module en soi, il regroupe un certain nombre de fonctions communes à mes modules perso.

J'ai opté pour cette solution pour rendre l'installation et la désinstallation plus aisée d'une part et pour éviter de mettre des choses dans les répertoires de XOOPS qui pourraient être modifiées ou supprimées lors d'une nouvelle version de XOOPS.

 

Fonctionalités type  
adminOnglet dossier Le repertoire onglet contient les fonctions de gestion des onglets de l'administration des modules.
Ces fonctionalités utilisent le template "adminOnglet.html"
spin dosier Ce repertoire contient un composant javaScript qui permet la saisie d'une valeur bornée par un minimum et un maximum.
editor_functions.php fichier Contient les fonctions de gestion des editeurs WYSIWYG. Prend en charge: KOIVI, InBetween, TinyEditor, HTMLArray, AHTMLArray
